# 初學者開發指引：從 Vibe Code 到 Git 檔案管理

## 任務目標
為初學者建立一份淺顯易懂的開發指南，解釋 VS Code、Google Antigravity、GitHub 以及 SourceTree 之間的關聯與差異，並引導使用者如何利用這些工具進行有效的檔案管理。

## 工具角色定義
1. **VS Code (工作台)**：你寫程式碼的地方，就像是你的筆記本。
2. **Google Antigravity (AI 助手/夥伴)**：你的智慧助手，可以幫你寫程式、修 Bug，甚至幫你操作 Git。
3. **Git (時光機)**：核心技術，負責紀錄檔案的所有更動紀錄。
4. **GitHub (雲端備份與社交)**：存放程式碼的雲端空間，讓你可以分享或與他人協作。
5. **SourceTree (視覺控制中心)**：利用圖形化界面讓你「看見」Git 的歷史紀錄，適合不習慣指令的初學者。
